# Getting started

Say goodbye to bad code and messy multi-repo compliance.
Using our CLI, you easily enforce best practices to organization-wide projects using global code compliances.

Exlint can be used on the command line to run, use and analyze code using customized policy groups.

:::tip

We don't need access to your code or anything whatsoever for you to setup Exlint in your projects!
:::

## Quickstart

### 1. Install the Exlint CLI

Binary releases can be downloaded from the [releases page](https://github.com/Exlint/CLI/releases) or can be installed directly

<Tabs>
  <TabItem value='Linux' label='Linux' default>

    curl https://get.datree.io | /bin/bash

  </TabItem>
  <TabItem value='MacOS' label='MacOS'>

    curl https://get.datree.io | /bin/bash

  </TabItem>
  <TabItem value='Windows (PowerShell)' label='Windows (PowerShell)'>

    iwr -useb https://get.datree.io/windows_install.ps1 | iex

  </TabItem>
  <TabItem value='Homebrew' label='Homebrew'>

    brew tap datreeio/datree
    brew install datreeio/datree/datree

  </TabItem>
  <TabItem value='Docker' label='Docker'>

    docker pull datree/datree

  </TabItem>
</Tabs>

### 2. Run a compliance check against your repo
Run the CLI against your code using a recommended compliance **or** if you already have a compliance set up, sync it to Exlint to run as a Group:

:::note
Exlint doesn't yet support every linter library.
Check out which libraries are [currently supported](#).
:::

**To run a recommended compliance:**

    exlint run -r

You will get the following output in your CLI:
*cli image*


**To sync your compliance as an Exlint Group:**

    exlint sync

Exlint will sync your compliance as a group and run it locally:
*image*

### 1. Customize your Group

Exlint helps you create multi-policy code compliances using **Groups** as compliances. Each Group can contain multiple linter library **Policies** with customzied configurations and **Rules**:&#x20;

#### Connect your CLI to your Exlint Dashboard
Use the following command to authenticate to your Exlint Dashboard

    exlint auth

To start using your dashboard, easily sign-up using your private **GitHub** or **Google** account, no need to enter any personal information.
