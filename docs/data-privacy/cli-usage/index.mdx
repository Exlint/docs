Exlint is committed to protecting the privacy of its users and their data.
The Exlint CLI is designed to analyze code and provide linting suggestions without collecting or transmitting any sensitive information.

When using the Exlint CLI, the only data that is transmitted is the metadata of the run results.
No personal information or identifiable data is collected or transmitted, and no user code is collected.

Additionally, all data transmitted to the Exlint application is protected using industry-standard encryption to ensure that it is kept secure.

If you have any concerns about the data privacy of the Exlint CLI, please contact us at contact@exlint.io.

You can check our privacy policy over [here](https://github.com/Exlint/usage-policies/blob/main/privacy-policy.mdx).
